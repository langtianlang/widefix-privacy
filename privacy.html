Privacy Policy for WideFix

Effective date: May 16, 2026

WideFix is a Chrome extension that improves the readability of websites on wide screens. All functionality runs locally in your browser.

Data Collection

WideFix does not collect, transmit, sell, or share personal information.

WideFix does not send browsing history, page content, URLs, or user activity to any server. The extension does not use external APIs, analytics, tracking services, or remotely loaded libraries.

No Backend Server

WideFix does not have a backend server. The extension does not send extension data, browsing data, page content, URLs, settings, or user activity to any server operated by WideFix or any third party.

Because there is no WideFix backend, there are no WideFix server logs, analytics logs, tracking logs, or remote databases for WideFix user activity.

Local Settings

WideFix stores only extension settings, such as your layout and readability preferences. These settings are stored locally in your browser using Chrome extension storage so the extension can remember your preferences.

Website Access

WideFix may read and modify the content of the webpage you are viewing only within your browser, and only to apply the readability and layout changes provided by the extension. This processing happens locally and is not transmitted anywhere.

Third Parties

WideFix does not share data with third parties.
